Las conexiones automáticas pueden afectar tanto al nivel de batería como al uso de datos si tus aplicaciones tienen acceso al servicio de datos.

Si seleccionas "Todo el tiempo" se consumirá aún más batería, pero el teléfono estará siempre actualizado.

Si optas por "Solo cuando esté usando el teléfono", SMARTACTIONS actualizará el teléfono cuando detecte movimiento o cuando la pantalla esté encendida.

La última opción, "Solo cuando esté usando la aplicación", es la que más batería ahorra, pero la aplicación se actualizará únicamente cuando se esté utilizando (sin recurrir a conexiones automáticas).
