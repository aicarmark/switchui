Os dados móveis permitem que seu dispositivo acesse a internet quando você não está conectado(a) ao Wi-Fi.

Ao ativar os dados móveis, o consumo da bateria e do plano de dados aumenta, mas eles são necessários para acessar a internet quando o Wi-Fi estiver indisponível.

Ao desativar os dados móveis, o consumo da bateria e do plano de dados é reduzido, mas isso o fará com que você precise de uma rede Wi-Fi para acessar a internet.
