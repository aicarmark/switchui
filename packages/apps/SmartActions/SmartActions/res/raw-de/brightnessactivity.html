Die Helligkeit des Displays lässt sich auf unterschiedliche Weise anpassen. Denken Sie daran, dass die Display-Helligkeit für die Akkulaufzeit des Telefons sehr wichtig ist.

Durch "Manuelle Helligkeit" können Sie das Display auf ein konstantes Helligkeitsniveau einstellen.

Durch "Automatische Helligkeit" wird das Display basierend auf den Lichtverhältnissen in der Umgebung angepasst. So können Sie den Displaytext stets lesen.

"Intelligente Helligkeit" ist mit "Automatische Helligkeit" vergleichbar. Dabei erfolgt eine Anpassung basierend auf dem Umgebungslicht, es werden jedoch niedrigere Helligkeitswerte verwendet, um den Akku zu schonen.
